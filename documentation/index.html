Folder & HTML Structure

'SoftEase' is a software / SaaS product HTML site template. So, you will need all .html files & supporting .css, .js, .php files with some images. Structure of your template would look like following image.

  • All .html files are in the root directory.
  • Separate folders are created for .css, .js files, for fonts & for images.
  • All fonts other than web-fonts are located in fonts folder.
  • Mailgun folder is for sending transactional emails.
  • Important .php files for sending emails are located in root folder.
What do you need to upload to server?

You will need to upload .html files along with supporting folders provided with this theme. Uploading complete HTML folder (provided with your download) would make it easy for you to setup the theme as shown on demo link.

General css classes:

  • .page-title: For page title div
  • .underline: For the line under header
  • .se-section: For symantic section on the page. Adds basic styling required for the section.
  • .se-feature: For displaying features
  • .container, .row, .col-*-*: Bootstrap classes for layout & grid. Read more on http://getbootstrap.com/
  • .mb*: For margin-bottom. Eg. mb1 for margin-bottom: 1em;
  • .mt*: For margin-top. Eg. mt1 for margin-top: 1em;
  • .pb*: For padding-bottom. Eg. pb1 for padding-bottom: 1em;
  • .pt*: For padding-top. Eg. pb2 for padding-top: 1em;
  • .btn.se-btn: For styled button

Contact Us Form Setup

We have integrated mailgun for uninterrupted transactional emails. You can use this free service for 10,000 emails every month. Visit mailgun.com for more information.

It is already integrated and ready to use. You just need to put your domain details & api-key in settings file.

  1. Create account on mailgun.com for free.
  2. You can use sandbox domain that you get or you can add your own domain with some simple steps provided at their website.
  3. You will need two important factors from your mailgun account. 1. Domain Name. 2. API Key
  4. Click on 'domains' link from navigation.
  5. Select the sandbox domain.
  6. On this screen on mailgun.com, you will get both details.
  7. Open mailgun-settings.php file which is included in your theme's downloaded folder.
  8. Paste mailgun domain name & API key here.
  9. Enter your email address in $yourEmail variable, where you want to receive emails from website. Replace support@themesease.com with your email id.
  10. You can change additional settings for email in same file. These are optional changes.

Make sure you have uploaded 'Mailgun' folder provided with your theme download.

Word Rotator Setup

We use textillate library for animating words. jQuery and animate.css files are required for this one to work. Make sure you have included both of them.

Add this line where you want to show word rotator:

<div class="white-color h1" id="animated-text">
    <ul class="texts">
        <li data-in-effect="fadeInUp" data-out-effect="fadeOutDown">Create fully responsive website.</li>
        <li data-in-effect="fadeInUp" data-out-effect="fadeOutDown">Top class template for software product.</li>
        <li data-in-effect="fadeInUp" data-out-effect="fadeOutDown">Specially designed for technical products.</li>
    </ul>
</div>

Following javascript code triggers word rotation:

// For Text Rotator
$('.animate-title').textillate({
	loop: true,
	type: 'word',
	in: {
		delay: 120
	},
	out: {
		delay: 120
	}
});

You can use different animations with these words easily. Refer http://textillate.js.org/ for more details.
